# Nina Carducci

Ce travail a été réalisé dans le cadre du projet 4 de la formation Développeur Web d’OpenClassrooms.

Ce repository contient l’optimisation du site vitrine de Nina Carducci, photographe professionnelle fictive.  
L’objectif principal était d’améliorer les performances, le référencement naturel (SEO) et l’accessibilité du site existant.

Le projet a impliqué l’analyse des performances, l’optimisation des images et la correction des problèmes d’accessibilité.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- SEO & Accessibilité (WCAG)
